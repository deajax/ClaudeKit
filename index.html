核心功能

切换、编码、调试、运行 —— 一个窗口，端到端的 Claude Code 工作体验。

内置终端，切换即开工

不用跳出应用。多标签终端内一键启动 Claude Code，切换模型商后即时生效。一边聊 Claude，一边跑 dev server，一个窗口搞定全部工作流。

多模型商一键切换

内置 DeepSeek、阿里云百炼、OpenRouter、硅基流动等主流模型商配置，下拉选择、一键应用，API 密钥与环境变量自动注入。

环境变量管理

图形化管理 Shell Profile 文件（.zshrc / .bashrc / PowerShell Profile），支持 Monaco 编辑器、语法高亮、敏感变量过滤。

任务管理

预设常用命令与参数模板，一键发送到终端标签页执行。npm dev、build、test 随时跑，减少重复输入，专注编码本身。

深色模式

精心调校的明暗双主题，一键切换，过渡流畅。终端、编辑器、所有 UI 面板同步适配。

余额查询

集成模型商余额查询 API，启动自动检测、手动刷新，底部状态栏实时显示。

技术架构

Electron + Vue 3 + TypeScript，全链路类型安全。

Electron

主进程负责 Pty 终端、文件 I/O、系统调用。通过 IPC 通道与前端通信。

Vue 3

组合式 API + TypeScript，Pinia 状态管理，Ant Design Vue 4.x UI 组件库。

JSON 存储

~/.ClaudeKit/ 目录下 JSON 文件持久化，不依赖网络，本地储存更安全。
